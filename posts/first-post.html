🛠️ 技术选型与架构

对于个人博客，我选择了简洁而高效的技术栈：

静态网站 + GitHub Pages

相比于复杂的CMS系统，静态网站有着显著的优势：

  • 加载速度快：无需数据库查询，纯静态文件
  • 维护成本低：没有服务器运维压力
  • 安全性高：静态文件很难被攻击
  • 版本控制：通过Git管理所有内容变更

现代化CSS设计

在视觉设计上，我采用了：

/* 毛玻璃效果 */
